Exercise 2

Although we may run out of time it would be cool to have you guys build a slack bot still. So write a python class with at least 1 function called talk which given a slack message will return a response. If your class has no response return NoneYou can code up this class to do whatever, return the current weather if someone asks for it, process a math equation etc. Just make sure that if your bot has a trigger phrase so what's the weather does nothing but hey weatherbot, what's the weather returns a response.

Below is a example if you wanted to build a weather bot 
> now
> weather bot, what's the weather tomorrow
it will be 23 degrees tomorrow in sydney
> weather bot, what's the weather tomorrow in chiagao
it will be -10 degrees tomorrow in chiago
...

You can add/remove as many features as you like but start small. Remember if the bot shouldn't respond to a message it should return none, the bot will recieve every message in the channel it is in so it needs to be able to ignore some messages.
